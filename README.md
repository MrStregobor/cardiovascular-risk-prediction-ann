# cardiovascular-risk-prediction-ann
# Heart Disease Prediction Using a Custom Artificial Neural Network

## Overview

This project implements a custom Artificial Neural Network (ANN) from scratch using NumPy to predict the presence of heart disease based on patient medical attributes. Unlike frameworks such as TensorFlow or PyTorch, the neural network's forward propagation, backpropagation, gradient descent optimization, and loss computation are manually implemented to demonstrate a deeper understanding of neural network fundamentals.

The model is trained and evaluated using the Heart Disease dataset and performs binary classification to determine whether a patient is likely to have heart disease.

---

## Features

* Custom ANN implementation using NumPy
* Manual forward and backward propagation
* Binary Cross-Entropy loss function
* Gradient Descent optimization
* Data preprocessing and feature standardization
* Training and testing dataset split
* Performance evaluation using:

  * Accuracy
  * Precision
  * Recall
  * F1-Score
  * Confusion Matrix
* Training loss visualization
* Performance metrics visualization

---

## Dataset

The model uses the Heart Disease dataset (`heart.csv`) containing clinical and demographic patient information.

### Input Features

* Age
* Sex
* Chest Pain Type (cp)
* Resting Blood Pressure (trestbps)
* Cholesterol (chol)
* Fasting Blood Sugar (fbs)
* Resting ECG Results (restecg)
* Maximum Heart Rate Achieved (thalach)
* Exercise-Induced Angina (exang)
* ST Depression (oldpeak)
* Slope of Peak Exercise ST Segment
* Number of Major Vessels (ca)
* Thalassemia (thal)

### Target Variable

* `0` → No Heart Disease
* `1` → Heart Disease Present

## Neural Network Architecture

| Layer          | Neurons | Activation |
| -------------- | ------- | ---------- |
| Input Layer    | 13      | -          |
| Hidden Layer 1 | 7       | Sigmoid    |
| Hidden Layer 2 | 7       | Sigmoid    |
| Hidden Layer 3 | 7       | Sigmoid    |
| Hidden Layer 4 | 7       | Sigmoid    |
| Output Layer   | 1       | Sigmoid    |

### Training Configuration

* Loss Function: Binary Cross-Entropy
* Optimizer: Gradient Descent
* Learning Rate: 0.1
* Epochs: 5000
* Random Seed: 42
